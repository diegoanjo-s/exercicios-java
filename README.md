## 🗂️ Exercícios

### Exercício 1 – Tabuada
**Enunciado:** O usuário entra com um número e o programa gera a tabuada de 1 até 10 desse número.

**Conceitos praticados:**
- Entrada de dados com `Scanner`
- Laço de repetição `for`
- Operações aritméticas

---

### Exercício 2 – Cálculo de IMC
**Enunciado:** O usuário informa sua altura e peso. O programa calcula o IMC (`peso / (altura * altura)`) e exibe o resultado com a classificação correspondente:

| IMC | Classificação |
|-----|--------------|
| ≤ 18,5 | Abaixo do peso |
| 18,6 – 24,9 | Peso ideal |
| 25,0 – 29,9 | Levemente acima do peso |
| 30,0 – 34,9 | Obesidade Grau I |
| 35,0 – 39,9 | Obesidade Grau II (Severa) |
| ≥ 40,0 | Obesidade III (Mórbida) |

**Conceitos praticados:**
- Entrada de dados com `Scanner`
- Operações aritméticas com `double`
- Estrutura condicional `if / else if / else`

---

### Exercício 3 – Par ou Ímpar no Intervalo
**Enunciado:** O usuário informa dois números (o segundo obrigatoriamente maior que o primeiro) e escolhe entre par ou ímpar. O programa lista todos os números pares ou ímpares dentro do intervalo informado.

**Conceitos praticados:**
- Entrada de dados com `Scanner`
- Laço `do...while` para validação de input
- Laço `for` para percorrer o intervalo
- Operador módulo `%` para verificar paridade

---

### Exercício 4 – Divisibilidade com Do While
**Enunciado:** O usuário informa um número inicial e depois continua informando outros números. O programa segue executando enquanto o número informado for divisível pelo número inicial. Números menores que o inicial são ignorados.

**Conceitos praticados:**
- Entrada de dados com `Scanner`
- Laço `do...while`
- Operador módulo `%` para verificar divisibilidade
- Estrutura condicional `if` para ignorar entradas inválidas
- Operador lógico `||` (OU)
