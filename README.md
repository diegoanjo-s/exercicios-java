# Exercícios Java

Exercícios práticos de Java desenvolvidos durante o Santander Bootcamp 2026.

## Exercício 1 - Tabuada
Programa que recebe um número do usuário e exibe a tabuada de 1 até 10.

**Conceitos praticados:** Scanner, loop for, operações aritméticas.

## Exercício 2 - Cálculo de IMC
Programa que recebe o peso e altura do usuário e calcula o IMC, exibindo a classificação correspondente.

**Conceitos praticados:** Scanner, double, int, conversão de tipos, if/else if.
